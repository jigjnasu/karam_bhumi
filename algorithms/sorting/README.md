## Time Complexity
| Algorithm | Worst Case | Average Case | Best Case | Memory |
|     :---      |     :---:      |     :---:      |     :---:      |     :---:      |
| Bubble   | $O(n^2)$ | $\theta(n^2)$ | $\omega(n)$ | $O(1)$ |
| Insertion   | $O(n^2)$ | $\theta(n^2)$ | $\omega(n)$ | $O(1)$ |
| Selection   | $O(n^2)$ | $\theta(n^2)$ | $\omega(n^2)$ | $O(1)$ |
| Heap   | $O(nlog_2n)$ | $\theta(nlog_2n)$ | $\omega(nlog_2n)$ | $O(1)$ |
| Quick   | $O(n^2)$ | $\theta(nlog_2n)$ | $\omega(nlog_2n)$ | $O(log_2n)$ |
| Merge   | $O(nlog_2n)$ | $\theta(nlog_2n)$ | $\omega(nlog_2n)$ | $O(n)$ |
